# ❄️ Smart Controlled Cooling Refrigerator UX

A collaborative university UX design project focused on improving refrigerator usability and reducing cold-air loss through user-centered design principles.

---

## 📖 Project Overview

Traditional refrigerators lose a significant amount of cold air whenever the main door is opened, resulting in increased energy consumption and reduced cooling efficiency.

This project proposes a Smart Controlled Cooling Refrigerator that minimizes cold-air loss through section-based access, an intuitive control interface and user-centered design.

---

## 🎯 Objectives

- Reduce cold-air loss
- Improve energy efficiency
- Improve usability
- Organize food storage
- Design a smarter refrigerator experience
